## Objective
- Build a "Best Paying Jobs in Every Country" website (bestpayingjobs.net) with 195 countries, 31 career categories, SEO-friendly URLs, OG images, salary comparison tools, and blog content.

## Important Details
- Salaries stored as **monthly** figures in each country's local currency (Nepal reference data in NPR, scaled to other countries via `target_avg_salary / nepal_avg_salary` then converted through USD to local currency using real exchange rates)
- URLs use `/best-paying-jobs-in-{slug}/` (hyphenated single segment) via `next.config.ts` rewrites in dev + `public/_redirects` for Cloudflare production — actual directory structure is nested `best-paying-jobs-in/[slug]`
- Dev server runs persistently via `nohup npx next dev -p 3000`
- Git repo at `git@github.com:yosintv2/bestpayingjobs.git`
- `public/og/` is in `.gitignore` (844MB images excluded); OG images must be regenerated on production server via `npm run generate-all`
- Shared `Header` and `Footer` components used across all pages — Header nav: Categories, Global Ranking, Calculators, Compare, Salary Letters, Blog
- Category page URL prefix is `/jobs/` — legacy `/category/` routes redirect via rewrites
- Global salary averages computed in USD using real exchange rates (`src/data/fx-rates.json`) — all `formatAnnual()` and `formatMonthly()` helpers output with `$` prefix
- Brand color palette: `emerald-600` primary, `emerald-700` hover, `emerald-50` light bg, `gray-900`→`gray-400` text hierarchy, `gray-200`/`gray-100`/`gray-50` borders/sections, `white` backgrounds
- Country pages have `id={category.slug}` anchors on each CategoryAccordion — category page links scroll via `/#{slug}`
- All 195 country flag images stored locally at `public/images/country/{slug}.png` (downloaded from singhs source with flagcdn.com fallback, validated as real PNGs via Content-Type check)
- Cost of living indices stored at `src/data/col-index.json` (195 countries, relative to NYC=100, used by `adjustedSalary()` in `src/lib/salary.ts`)
- City dataset at `src/data/cities.json` (691 cities — capitals + major cities, generated by `scripts/generate-cities.mjs`)
- City pages at `/salary-in-{slug}/` via rewrite to `/salary-in/{slug}` — 691 static pages showing top 10 city-scaled jobs + OG image + link to country page
